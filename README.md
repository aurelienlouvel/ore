# oré

Portfolio website — monorepo with Next.js frontend and Sanity Studio CMS.

## Structure

```
├── nextjs-app/       # Next.js 16 + Tailwind CSS
└── sanity-studio/    # Sanity Studio v3
```

## Getting started

```bash
# Next.js app
cd nextjs-app && pnpm install && pnpm dev

# Sanity Studio
cd sanity-studio && pnpm install && pnpm dev
```
